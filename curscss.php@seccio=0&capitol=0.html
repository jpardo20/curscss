Secció 0: Introducció

Descripció de la secció

La secció 0 introdueix diversos aspectes relacionats amb els style sheets. Si ho prefereixes, pots passar directament a la secció 1, que és on en realitat comença el curs, però jo et recomanaria llegir aquests apartats, si més no per assegurar-te que ja coneixes els conceptes que hi apareixen.

Primer es presenta una introducció a la relació entre CSS, HTML i Internet, així com un brevíssim resum de l'evolució del disseny web. Després tractarem alguns aspectes concrets que et poden ajudar al llarg d'aquest curs, com ara la diferència entre element i tag o els conceptes d'element de bloc i element inline. El darrer apartat presenta els signes convencionals que s'utilitzen al llarg d'aquest curs.

  1. HTML i CSS
  2. Especificacions: el W3C
  3. Element vs. Tag
  4. Elements de bloc, elements inline
  5. Signes convencionals
