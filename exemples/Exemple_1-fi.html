<STYLE> <!--

/* =====================================
   Exemple elemental de style sheet
   Autor: Gonzalo Pérez de la Ossa
   =====================================*/

/* Aquest style sheet està sobrecarregat de
   comentaris, només per servir d'exemple. */


BODY {
margin: 30px;
font-family: Verdana; font-size: 15px;
background-color: #EEF2FF; }

/* Jo acostumo a agrupar en una línia les propietats
   que estan relacionades, però recordeu separar-les
   sempre per punt i coma. */



H1 {
font-family: Arial; color: navy;
text-align: center; }



P {                  /* La propietat text-indent permet sagnar */
text-indent: 20px; } /* la primera línia de cada paràgraf */



/* P i H1 tindran també les propietats definides pel selector
   BODY, excepte la font dels títols H1, que està sobreescrita. */

--> </STYLE>

Et proposo a continuació un document HTML que utilitza aquest CSS, però seria millor si tu mateix intentessis fer-ne un de nou. Fins i tot podries intentar crear el teu propi style sheet i així experimentar quin resultat obtens aplicant els diferents estils. Ah, només escric el BODY, però segur que no tens cap dubte sobre com incorporar-hi l'style sheet. Si cal, revisa la Secció 1 apartat 3, 'Associar style sheets a un document HTML: Linking i Embedding'.
