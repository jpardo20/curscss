CSS en català Exemple 2-2   Selectors pseudoclasse: links

Et recordo quin era el contingut de l'style sheet:

A { text-decoration: none }
:visited { color: olive }
:active { color: green }

:hover { text-decoration: underline }

A.menu:hover {
text-decoration: underline overline }

P#titol A:hover {
border-style: solid; border-width:1px; padding:4px;
text-decoration: none }


I aquest és el codi HTML que veuràs:

<P id="titol">
   Això és un element P amb id="titol".
   Conté un <A href="">link</A>.
</P>

<DIV align="center">
   <A href="" class="menu">inici</A> |
   <A href="" class="menu">contacte</A> |
   <A href="" class="menu">links</A>
</DIV>

<P>I per acabar, un paràgraf amb links 'normals',
com ara <A href="">aquest</A>.</P>



I finalment, el resultat obtingut:

Això és un element P amb id="titol". Conté un link.


inici | contacte | links

I per acabar, un paràgraf amb links 'normals', com ara aquest.
