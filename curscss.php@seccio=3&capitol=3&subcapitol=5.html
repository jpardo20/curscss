3. Propietats relacionades amb el text

3.5. font-variant

Permet especificar l'ús de small caps ('majúscules petites') per a les lletres minúscules. Els valors possibles són normal i small-caps.

La delaració

H1 {
font-family: "Courier New";
font-variant: samll-caps; }
