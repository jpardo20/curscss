3. Propietats relacionades amb el text

3.13. letter-spacing

Estableix la separació entre els caràcters del text. El valor s'ha d'expressar mitjançant longituds. El valor per defecte és normal, i el defineix el navegador.

H1 { letter-spacing: 0.1em; }

Els valors poden ser negatius, tot i que sovint obtindrem un text amb les lletres superposades.
